# 规范

放当前长期有效的项目规范、协议和架构说明。

这里的文档用于指导后续实现。方案落地后，如果规则会长期存在，应从 `superpowers/` 或 `review/` 同步沉淀到这里。

当前文档：

| 文档 | 用途 |

| `REVERSE_EXPORT.md` | InDesign 反向导出、观察模式和语义化迁移规范 |
| `FONT_POLICY.md` | 项目公共字体库、字体 token、映射和预检规范 |
