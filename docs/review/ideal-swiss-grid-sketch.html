真正的网格先行：元素只用行列索引定位

理想布局网格

这版草图把网格定义和内容摆放彻底分开：页面先定义固定边距、12 列、8 个等高行模块、固定列缝和固定行缝；内容区域只能用行列起点和跨度生成。

因此，参考线不是从对象边缘提取出来的。反过来，对象边缘必须贴合这些先验网格线。

紫色：页边距 / 安全框定义内容区边界；进入 InDesign margin 与页面参考线。
蓝色：12 列均质主网格左右位置只由列起点和跨列数量决定；列宽全部相同。
深蓝：固定列缝列缝是不可放置区域，避免图文压进 gutter。
橙色：8 个等高行模块上下位置只由行起点和跨行数量决定；不为元素单独加线。
浅橙：固定行缝行模块之间的呼吸距离，和列缝一样先验存在。
灰色虚线：局部 baseline只在文字区域内表达文字节奏，不作为全页排版网格。
内容框：由行列索引计算示例内容没有手写坐标；所有位置来自 col / row / span。
