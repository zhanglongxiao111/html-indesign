02 核心与观众区

跨列数量由设计网格决定。

03 运营配套

上下排卡片对齐粗行模块。

04 图纸与指标

4mm baseline 不直接变成满屏参考线。

示例标注线可不贴主网格；它属于 annotation 层。

网格线草图

这张图区分“控制版面的主设计网格”和“只用于文字/间距节奏的细 baseline”。推荐导出到 InDesign 的是主设计网格，不是每一条细 baseline。

紫色：页边距 / 安全框定义内容可进入的页面范围；应映射为 InDesign margin。
蓝色：12 列主网格控制大块图文、卡片、图纸框的左右位置和跨列宽度；应导出为 ID 参考线。
深蓝：栏间距两侧表达列之间的 gutter 边界；用于避免对象落在栏缝中。
橙色：粗行模块控制页面纵向大区块，例如标题区、图纸区、卡片行、页脚区；应导出为 ID 参考线。
灰色虚线：baseline / 间距节奏用于文字行距、padding、局部间距；默认不应全量导出为 ID 参考线。
红色：示例对象边缘对象应贴合主列和粗行模块；对象边缘不是网格来源。
