## 4. 自由页规范 (Free Page)

当使用 `F-自由页` 母版时，可以动态创建元素：

```html
<section data-template="free" data-master="F-自由页">
  <!-- 填充母版槽位 -->
  <div data-slot="分页标题">页面标题 / Page Title</div>
  
  <!-- 动态创建文本框 -->
  <div class="free-item" data-type="TEXT"
       data-paragraph-style="小节标题（36点左对齐）"
       data-bounds="15mm,30mm,200mm,20mm">
    文本内容
  </div>
  
  <!-- 动态创建图片框 -->
  <div class="free-item" data-type="IMAGE"
       data-object-style="[基本图形框架]"
       data-bounds="15mm,55mm,180mm,120mm">
    <img src="image.jpg"/>
  </div>
</section>
```

### data-bounds 格式

`data-bounds="x,y,width,height"` - 所有值使用 mm 单位

- `x`: 元素左边距到页面左边的距离
- `y`: 元素上边距到页面上边的距离
- `width`: 元素宽度
- `height`: 元素高度
